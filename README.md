## 🛠️ 開始使用

請依照以下步驟在您的本機環境架設此專案：

### 前置作業
* 已安裝 Python 3.8 或以上版本。
* 已安裝 Git。

### 安裝步驟

1. **複製專案**
   ```bash
   git clone [https://github.com/](https://github.com/){您的帳號}/{專案名稱}.git
   cd {專案名稱}

2. 建立虛擬環境 (建議)
    python -m venv venv
    # Windows 啟動:
    venv\Scripts\activate
    # Mac/Linux 啟動:
    source venv/bin/activate
3. 安裝依賴套件
    pip install -r requirements.txt
4. 執行程式
    python main.py
